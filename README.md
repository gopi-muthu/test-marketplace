# Forte3D Copilot Plugins

GitHub Copilot agent plugins for the **Forte3D / S3Dx** development team. This repository contains
domain-specific skills that extend Copilot with knowledge of S3Dx client development, test
automation, and Azure DevOps workflows.

## Plugins

### [Forte3D](plugins/forte3d/README.md)

GitHub Copilot skills for Forte3D / S3Dx client development workflows. Includes C# command
generation (dialog and NoUI patterns), S3Dx XML command registration, automated test (MCAT)
creation and management, and Azure DevOps pull request code review.

### [Trekkers](plugins/trekkers/README.md)

GitHub Copilot skills for Trekkers team development workflows. Includes MCAT automation coverage
reporting with Azure DevOps cross-referencing.
