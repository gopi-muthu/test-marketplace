# Trekkers Plugin

GitHub Copilot skills for **Trekkers** team development workflows. Includes MCAT automation
coverage reporting with Azure DevOps cross-referencing.

## What's Included

### Skills

| Skill | Description |
| ----- | ----------- |
| `mcat-sanity-check` *(preview)* | Generate or refresh a comprehensive MCAT automation status report by scanning the codebase and cross-referencing test case coverage with Azure DevOps. |
